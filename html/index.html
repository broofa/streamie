Streaming Frame tests

Frame/htmlfile stream
The text/html content stream for iframe and htmlfile-based APIs. Contains SCRIPT fragments that generate JSONP calls.
Frame/htmlfile stream -> IFrame
The Frame/htmlfile stream piped into an iframe. Probably not terribly interesting since it's basically the same mechanics as the bare Frame/htmlfile stream above. A stepping stone to the actual ActiveXObject('htmlfile')-based implementation I'll be implementing a test for shortly.

Streaming XHR/XDR tests

XDR/XHR stream
The content stream for use by XDomainRequest and XMLHttpRequest APIs. \n-separated JSON messages. Note that this Content-Type for this is application/octet-stream to work around a Chrome caching issue, so this will likely show up as a file download.
XDR/XHR stream -> XDomainRequest (IE only)
Test of the XDR/XHR stream fed into an XdomainRequest object. (view source for details)
XDR/XHR stream -> XMLHttpRequest
Test of the XDR/XHR stream fed into an XMLHttpRequest object. (same source as the above, but with the 'xhr' param set)
